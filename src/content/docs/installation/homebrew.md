## Homebrew on MacOS

Install using Homebrew:

```bash
brew install feroxbuster
```

## Homebrew on Linux via tap

[Linux tap](https://github.com/TGotwig/homebrew-linux-feroxbuster/blob/main/feroxbuster.rb)

```bash
brew tap tgotwig/linux-feroxbuster
brew install feroxbuster
```
